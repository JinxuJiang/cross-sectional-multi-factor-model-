# 项目愿景 (Project Vision)
1. 本项目旨在构建一个可复现、可维护、可快速迭代的截面多因子量化选股系统。通过引入 AI 模型（LightGBM/XGBoost/Transformer）捕捉因子与收益率之间的非线性关系，实现超越基准指数的超额收益。
2. 系统架构 (System Architecture)
    整个系统分为四个核心模块，遵循“存算分离”和“低耦合”原则：
    Module 1: Data Engine (数据引擎)
    负责原始行情（OHLCV）与财务数据（PIT格式）的搬运与存储。
    存储格式：本地 Parquet（按股票代码索引），兼顾读取速度与压缩率。
    Module 2: Alpha Factory (因子工厂)
    特征工程核心。负责因子计算（算子化）、数据清洗（去极值、标准化）、中性化处理。
    实现“Point-in-Time”对齐，杜绝未来函数。
    Module 3: Alpha Combiner (AI 训练与集成)
    使用 AI 模型对多维度因子进行权重分配与非线性建模。
    包含 Walk-forward（滚动训练）逻辑，防止过拟合。
    Module 4: Optimizer & Backtest (组合优化与回测)
    将模型输出的预测分数转化为交易头寸。
    考虑行业中性、换手率约束及交易成本。

4. 数据标准 (Data Specification)
    行情数据标准：
    频率：日频 (1d)
    复权：前复权 (Front Adjustment)
    列名：time, open, high, low, close, volume, amount
    财务数据标准：
    格式：PIT (Point-in-Time)
    对齐键：m_anntime (公告日), report_date (报告期)
    元数据标准：
    行业分类：申万一级行业
    股票池：全A股 / 沪深300 / 中证500/ 中证1000
    数据检验标准：
    分层检验：原始数据层 + 处理后数据层
    检验类型：完整性、一致性、业务规则
    报告格式：Markdown 报告，包含通过/失败状态
5. AI 赋能路线图 (AI Roadmap)

    阶段 2: 遗传算法自动因子挖掘

    阶段 3: LightGBM 截面回归模型实现
