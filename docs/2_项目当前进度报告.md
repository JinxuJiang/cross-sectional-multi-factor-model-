- **PIT缝合**: 合并多张财务表（Income/Balance/CashFlow/Indicator），按报告期前向填充
- **等比前复权**: 使用乘法公式调整，避免传统前复权的负数问题

### 2.3 技术亮点

- **"一家族一文件"设计**: 按因子家族聚合，而非一因子一文件
- **因果逻辑Docstring**: 每个因子包含经济学解释和参考文献
- **向量化计算**: 使用NumPy矩阵运算，高效处理大规模数据
- **Pipeline清洗流程**: 
  ```
  原始因子 → MAD去极值(3倍MAD) → 填补缺失(行业中位数) → OLS中性化 → Z-Score标准化 → N(0,1)
  ```

### 4.3 回测流程

```
1. 加载预测数据 (predictions.parquet + live_predictions.parquet)
2. 加载行情数据 (open/high/low/close/volume 宽表)
3. 合并数据 → 长表格式
4. 信号生成:
   - 按调仓周期(默认20日)筛选日期
   - 每期货选预测分数最高的N只股票
   - 生成买入/卖出列表
5. Backtrader回测:
   - 初始资金: 100,000
   - 手续费: 0.2% (双边)
   - 目标仓位: 90% / 持仓数量
6. 输出绩效报告 + 净值曲线
```

## 六、已实现的因子列表

### 6.1 技术因子 (Technical Factors)

| 因子名 | 描述 | 因果逻辑 |
|--------|------|----------|
| ret20 | 20日收益率 | 短期动量，反映市场情绪和资金流入 |
| ret60 | 60日收益率 | 中期动量，反映趋势持续性 |
| ret120 | 120日收益率 | 长期动量，反映基本面改善预期 |
| ret20_60 | 动量差 (ret20 - ret60) | 动量加速度，识别加速/减速趋势 |

### 6.2 财务因子 (Financial Factors)

| 类别 | 因子名 | 描述 |
|------|--------|------|
| 估值 | PE | 市盈率 TTM |
| 估值 | PB | 市净率 |
| 估值 | PS | 市销率 TTM |
| 盈利 | ROE | 净资产收益率 TTM |
| 盈利 | ROA | 总资产收益率 TTM |
| 质量 | 周转率 | 资产周转率 |
| 质量 | 现金流 | 经营现金流/营业收入 |
| 成长 | 营收增长 | 营业收入YoY |
| 成长 | 利润增长 | 净利润YoY |
