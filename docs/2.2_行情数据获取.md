## 二、核心 API 说明

### 2.1 下载历史数据

```python
xtdata.download_history_data(
    stock_code='000001.SZ',      # 合约代码
    period='1d',                  # 周期: 1d=日线, 1m=1分钟
    start_time='20240101',        # 起始时间 (YYYYMMDD)
    end_time='',                  # 结束时间 (空=最新)
    incrementally=None            # 是否增量下载
)
```

**批量版本**:
```python
xtdata.download_history_data2(
    stock_list=['000001.SZ', '600000.SH'],
    period='1d',
    start_time='20100101',
    callback=on_progress          # 进度回调函数
)
```

### 2.2 获取市场数据

```python
data = xtdata.get_market_data_ex(
    field_list=[],                # 字段列表 (空=全部)
    stock_list=['000001.SZ'],     # 股票列表
    period='1d',                  # 周期
    start_time='20100101',        # 起始时间
    dividend_type='front_ratio'   # 复权方式
)
```

**复权方式选项**:
| 类型 | 说明 | 是否推荐 |
|------|------|---------|
| `none` | 不复权 | ❌ |
| `front` | 前复权（加减法）| ❌ 可能出现负数 |
| `back` | 后复权 | ❌ |
| **`front_ratio`** | **等比前复权（乘法）** | ✅ **推荐** |
| `back_ratio` | 等比后复权 | ❌ |

## 四、数据格式

### 4.1 单只股票存储格式

**文件**: `market_data/000001.SZ.parquet`

| 字段 | 类型 | 说明 |
|------|------|------|
| `time` | datetime.date | 交易日期 |
| `open` | float | 开盘价 |
| `high` | float | 最高价 |
| `low` | float | 最低价 |
| `close` | float | 收盘价（等比前复权）|
| `volume` | int | 成交量（股）|
| `amount` | float | 成交额（元）|
| `preClose` | float | 前收盘价 |

### 4.2 读取数据示例

```python
import pandas as pd

# 读取单只股票
df = pd.read_parquet('数据/data/raw_data/market_data/000001.SZ.parquet')
print(df.tail())

# 输出示例:
#               time   open   high    low  close    volume      amount  preClose
# 2024-01-01  10.20  10.35  10.15  10.30  15000000  154500000.0     10.15
```

## 六、增量更新

### 6.1 检查本地最新日期

```python
import pandas as pd
import os
from datetime import datetime, timedelta

def get_last_update_date(stock_code):
    """获取某只股票本地最新数据日期"""
    file_path = f'data/raw_data/market_data/{stock_code}.parquet'
    if not os.path.exists(file_path):
        return None
    df = pd.read_parquet(file_path)
    return df['time'].max()
```

### 6.2 增量下载逻辑

```python
def incremental_download(stock_list):
    for stock in stock_list:
        last_date = get_last_update_date(stock)
        if last_date:
            # 从最后一天+1开始下载
            start = (last_date + timedelta(days=1)).strftime('%Y%m%d')
        else:
            start = '20100101'  # 全量下载
        
        xtdata.download_history_data(stock, '1d', start)
```

## 七、注意事项

### 7.1 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 连接失败 | QMT 未启动/权限不足 | 检查投研端权限，确认行情服务器地址 |
| 下载超时 | 股票数量过多 | 分批下载，每批 300 只 |
| 数据缺失 | 停牌/退市 | 检查 suspendFlag，跳过缺失数据 |
| 负数价格 | 使用普通前复权 | 改用 `front_ratio` 等比前复权 |

### 7.2 性能优化

- **批量下载**: 使用 `download_history_data2` 而非循环单只下载
- **本地缓存**: API 会自动缓存到本地，重复获取时更快
- **并发控制**: 避免同时下载过多股票，建议每批 300 只以内
