# 2.1 数据层总体要求

---

## 一、数据层定位

**模块归属**: Module 1 - Data Engine (数据引擎)

**核心职责**:
- 从迅投 QMT 下载原始行情数据和财务数据
- 建立统一、结构清晰、可维护的数据仓库
- 为上层因子工厂提供标准化数据输入

**设计原则**:
- **存算分离**: 数据存储与计算逻辑解耦
- **增量更新**: 支持每日增量下载，避免全量重复下载
- **数据校验**: 每一层数据都有对应的检验机制

---

## 二、数据来源

### 2.1 迅投 QMT API

**使用接口**:
- `xtdata.download_history_data()` / `download_history_data2()` - 下载行情数据
- `xtdata.get_market_data_ex()` - 获取行情数据（等比前复权）
- `xtdata.download_financial_data2()` - 下载财务数据
- `xtdata.get_financial_data()` - 获取财务数据（PIT格式）

**连接方式**:
- 投研端原生 Python API（无需启动 QMT 客户端界面）
- 需要投研端 VIP 权限连接行情服务器

### 2.2 数据分类

| 数据类型 | API来源 | 更新频率 | 存储格式 |
|---------|---------|---------|---------|
| **行情数据** | `get_market_data_ex` | 日频 | Parquet (个股文件) |
| **财务数据** | `get_financial_data` | 季度 | Parquet (个股文件) |
| **行业数据** | `get_stock_list_in_sector` | 月度 | CSV |
| **元数据** | `get_instrument_detail` | 不定期 | Parquet |

## 四、数据获取流程

### 4.1 首次全量下载

```
1. 连接 QMT 行情服务器
   └─→ xtdata.connect(ip, port)

2. 获取股票列表
   └─→ xtdata.get_stock_list_in_sector('沪深A股')

3. 批量下载行情数据
   └─→ download_history_data2(stock_list, period='1d', start_time='20100101')
   └─→ get_market_data_ex(dividend_type='front_ratio')  # 等比前复权

4. 批量下载财务数据
   └─→ download_financial_data2(stock_list)
   └─→ get_financial_data(report_type='announce_time')  # 按公告时间

5. 下载行业与元数据
   └─→ get_all_industry_map()      # 申万一级行业
   └─→ get_all_stock_details()     # 股票基础信息
```

### 4.2 每日增量更新

```
1. 检查本地最后更新日期
   └─→ 读取 download_log.json

2. 增量下载行情数据
   └─→ download_history_data(incrementally=True)

3. 检查财务数据更新（财报季）
   └─→ 对比已有数据，补充新公告的财报

4. 更新日志记录
   └─→ 更新 download_log.json
```

## 五、关键技术要点

### 5.1 复权方式选择

**使用等比前复权 (`front_ratio`)**:
- 原理: `复权价格 = 原始价格 × 复权系数`（乘法）
- 优点: 不会出现负数价格（传统前复权可能出现）
- 适用: 长期回测、收益率计算

### 5.2 分批下载机制

- **行情数据**: 每批 300 只股票（API限制）
- **财务数据**: 每批 300 只股票（避免超时）
- **延迟**: 批次间 sleep 5-10 秒，避免触发限流

### 5.3 错误处理与重试

- **网络异常**: 自动重试 3 次
- **缺失数据**: 记录缺失股票代码，事后补录
- **格式异常**: 捕获异常，跳过该股票，继续处理

## 六、数据检验要求

### 6.1 完整性检验

- 检查是否所有股票都已下载
- 检查时间范围是否完整（是否有断档）
- 检查字段完整性（OHLCV 是否齐全）

### 6.2 一致性检验

- 股票代码格式统一（000001.SZ / 600000.SH）
- 时间戳格式统一（datetime.date）
- 行情数据与财务数据的时间范围对齐

### 6.3 业务规则检验

- 价格不能为负数（等比前复权后检查）
- 成交量不能为负数
- 停牌股票标记（suspendFlag）

## 八、关键代码示例

### 8.1 下载单只股票行情

```python
from xtquant import xtdata

# 下载数据
xtdata.download_history_data('000001.SZ', '1d', '20100101')

# 获取等比前复权数据
data = xtdata.get_market_data_ex(
    field_list=['open', 'high', 'low', 'close', 'volume'],
    stock_list=['000001.SZ'],
    period='1d',
    dividend_type='front_ratio'  # 等比前复权
)
df = data['000001.SZ']
```

### 8.2 获取全市场股票列表

```python
# 获取沪深A股全列表
all_stocks = xtdata.get_stock_list_in_sector('沪深A股')
print(f"全市场共 {len(all_stocks)} 只股票")
```
