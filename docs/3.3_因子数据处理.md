# 3.3 因子数据处理 (Processors)

---

## 一、概述

**模块归属**: Module 2 - Alpha Factory (因子工厂) - 数据清洗子模块

**核心职责**: 对计算完成的因子进行标准化清洗，使其具备可比性

**处理原则**: 
- **4步标准流程**: 去极值 → 填补缺失 → 中性化 → 标准化
- **截面处理**: 每天独立处理，不跨天使用数据
- **直接覆盖**: 清洗后的因子直接覆盖原文件

---

## 二、四步清洗流程

```
原始因子值
    │
    ▼
┌─────────────┐
│ 1. 去极值   │  ← MAD算法，3倍中位数绝对偏差
│ (Outlier)   │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 2. 填补缺失 │  ← 剔除无效数据，行业/市值缺失的drop
│  (Missing)  │     其他缺失用行业中位数填补
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 3. 中性化   │  ← OLS回归，剥离行业和市值Beta
│(Neutralize) │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 4. 标准化   │  ← Z-Score，输出N(0,1)分布
│(Standardize)│
└──────┬──────┘
       │
       ▼
清洗后因子值
```

## 三、Step 1: 去极值 (Outlier Removal)

### 3.1 MAD算法

**MAD** = Median Absolute Deviation（中位数绝对偏差）

**公式**:
```
MAD = median(|x_i - median(x)|)

上下界:
  upper = median(x) + n_mad × MAD
  lower = median(x) - n_mad × MAD

默认: n_mad = 3.0（3倍MAD）
```

### 3.2 代码实现

```python
def mad_winsorize(factor: pd.Series, n_mad: float = 3.0) -> pd.Series:
    """
    MAD去极值
    
    参数:
    -----
    factor : pd.Series
        原始因子值，index=股票代码
    n_mad : float
        MAD倍数，默认3.0
    
    返回:
    ------
    pd.Series : 去极值后的因子
    """
    median = factor.median()
    mad = (factor - median).abs().median()
    
    upper = median + n_mad * mad
    lower = median - n_mad * mad
    
    # 截断处理
    return factor.clip(lower=lower, upper=upper)
```

### 3.3 与3-sigma的区别

| 方法 | 优点 | 缺点 |
|------|------|------|
| **MAD** | 对极端值不敏感，更稳健 | 计算稍复杂 |
| **3-sigma** | 简单直观 | 受极端值影响大 |

**推荐**: 量化中普遍使用 MAD，更稳健

## 四、Step 2: 填补缺失值 (Missing Value Imputation)

### 4.1 缺失类型与处理策略

| 缺失类型 | 处理方式 | 原因 |
|---------|---------|------|
| **因子值缺失** | 行业中位数填补 | 保持截面完整性 |
| **行业缺失** | 直接剔除 (drop) | 无法分类，无法中性化 |
| **市值缺失** | 直接剔除 (drop) | 无法市值中性化 |

### 4.2 代码实现

```python
def fill_missing(factor: pd.Series, 
                 industry: pd.Series, 
                 market_cap: pd.Series) -> tuple:
    """
    填补缺失值
    
    策略:
    -----
    1. 剔除行业或市值缺失的股票
    2. 因子缺失的用行业中位数填补
    """
    # 合并数据
    df = pd.DataFrame({
        'factor': factor,
        'industry': industry,
        'market_cap': market_cap
    })
    
    # 剔除行业或市值缺失
    df_clean = df.dropna(subset=['industry', 'market_cap'])
    
    # 因子缺失的用行业中位数填补
    for industry_name in df_clean['industry'].unique():
        mask = df_clean['industry'] == industry_name
        median_value = df_clean.loc[mask, 'factor'].median()
        df_clean.loc[mask, 'factor'] = df_clean.loc[mask, 'factor'].fillna(median_value)
    
    return df_clean['factor'], df_clean['industry'], df_clean['market_cap']
```

## 五、Step 3: 中性化 (Neutralization)

### 5.1 中性化目标

**剥离已知Beta风险，保留纯Alpha**：
- **行业Beta**: 不同行业的因子水平差异（如银行业PE普遍低）
- **市值Beta**: 大小市值股票的因子水平差异（如小盘股波动大）

### 5.2 OLS回归法

```
因子 = β₀ + β₁ × 行业dummy + β₂ × ln(市值) + ε

中性化后因子 = ε（残差）
```

**步骤**:
1. 对行业做one-hot编码（dummy变量）
2. 对市值取对数 `ln(market_cap)`
3. OLS回归，提取残差
4. 残差即为中性化后的因子

### 5.3 代码实现

```python
def neutralize(factor: pd.Series, 
               industry: pd.Series, 
               market_cap: pd.Series) -> pd.Series:
    """
    OLS中性化：剥离行业和市值Beta
    
    参数:
    -----
    factor : pd.Series - 原始因子
    industry : pd.Series - 行业分类（申万一级）
    market_cap : pd.Series - 市值数据
    
    返回:
    ------
    pd.Series - 中性化后的因子（残差）
    """
    import statsmodels.api as sm
    
    # 构建回归矩阵
    X = pd.get_dummies(industry, prefix='ind')  # 行业dummy
    X['ln_cap'] = np.log(market_cap)            # 对数市值
    X = sm.add_constant(X)                      # 添加常数项
    
    # OLS回归
    model = sm.OLS(factor, X).fit()
    
    # 返回残差（中性化后因子）
    return model.resid
```

### 5.4 为什么只中性化行业和市值？

| 风险因子 | 是否中性化 | 原因 |
|---------|-----------|------|
| **行业** | ✅ 是 | 已知且稳定的Beta |
| **市值** | ✅ 是 | 最显著的风格因子 |
| **估值** | ❌ 否 | 本身就是要挖掘的Alpha |
| **动量** | ❌ 否 | 本身就是要挖掘的Alpha |

## 六、Step 4: 标准化 (Standardization)

### 6.1 Z-Score标准化

**公式**:
```
z = (x - μ) / σ

其中:
  μ = mean(x)  # 均值（期望为0）
  σ = std(x)   # 标准差（期望为1）
```

### 6.2 代码实现

```python
def zscore_standardize(factor: pd.Series) -> pd.Series:
    """
    Z-Score标准化
    
    输出: 均值为0，标准差为1的分布
    """
    mean = factor.mean()
    std = factor.std()
    
    if std == 0:
        return pd.Series(0, index=factor.index)
    
    return (factor - mean) / std
```

### 6.3 为什么用Z-Score而非Rank？

| 方法 | 优点 | 适用场景 |
|------|------|---------|
| **Z-Score** | 保留因子间的距离信息 | 因子值分布近似正态 |
| **Rank** | 对极端值不敏感 | 因子值分布严重偏斜 |

**推荐**: 一般先用Z-Score，如遇极端偏斜再考虑Rank

### 7.3 Pipeline内部流程

```python
def clean_factor(factor, industry, market_cap, steps=None, verbose=False):
    """一键清洗因子（单截面）"""
    if steps is None:
        steps = ['outlier', 'missing', 'neutralize', 'standardize']
    
    result = factor.copy()
    
    # Step 1: 去极值
    if 'outlier' in steps:
        result = mad_winsorize(result, n_mad=3.0)
    
    # Step 2: 填补缺失
    if 'missing' in steps:
        result, industry_clean, market_cap_clean = fill_missing(
            result, industry, market_cap
        )
    
    # Step 3: 中性化
    if 'neutralize' in steps:
        result = neutralize(result, industry_clean, market_cap_clean)
    
    # Step 4: 标准化
    if 'standardize' in steps:
        result = zscore_standardize(result)
    
    return result
```

## 九、注意事项

### 9.1 处理顺序不可变

**必须按顺序**: 去极值 → 填补缺失 → 中性化 → 标准化

**原因**:
- 中性化前必须填补缺失（OLS不能处理NaN）
- 标准化必须最后（确保输出N(0,1)）

### 9.2 截面独立性

**每天独立处理**，不能跨天使用数据：
- 不能用前一天的均值填补今天
- 不能用历史数据计算今天的MAD

**原因**: 防止引入未来函数，确保回测真实性

### 9.3 样本剔除

中性化时会剔除无行业/无市值的股票，这会导致：
- 清洗后样本数 ≤ 清洗前样本数
- 这是正常的，这些股票无法用于截面选股

## 十、完整示例

```python
import pandas as pd
from processors.pipeline import clean_factor, clean_factor_summary

# 构造测试数据
np.random.seed(42)
stocks = [f"{i:06d}.SZ" for i in range(1, 101)]

# 行业
industries = ['银行'] * 20 + ['医药'] * 20 + ['科技'] * 20 + ['消费'] * 20 + ['能源'] * 20
industry = pd.Series(industries, index=stocks)

# 市值（对数正态分布）
market_cap = pd.Series(np.random.lognormal(20, 0.5, 100), index=stocks)

# 原始因子（含极端值和缺失）
factor_raw = pd.Series(
    np.where(np.array(industries) == '银行', 2.0, 0.0) + np.random.normal(0, 1, 100),
    index=stocks
)
factor_raw.iloc[0] = 100   # 极端大值
factor_raw.iloc[1] = -50   # 极端小值
factor_raw.iloc[95:100] = np.nan  # 缺失值

# 一键清洗
factor_clean = clean_factor(
    factor_raw, 
    industry, 
    market_cap, 
    verbose=True
)

# 输出统计
print(f"\n清洗后均值: {factor_clean.mean():.6f} (应≈0)")
print(f"清洗后标准差: {factor_clean.std():.4f} (应≈1)")
```
